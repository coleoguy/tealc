# External Peer Review Template

## Instructions for reviewers

Thank you for participating in this blinded evaluation. You have been provided with:

1. A `.jsonl` file containing blinded output items — one JSON object per line.
2. A `_manifest.json` file with batch details and a pointer to the domain rubric.
3. This review template to fill in and return.

For each item in the `.jsonl` file, complete one score block below. Copy the `blinded_id` value exactly from the JSON object. Score each dimension on an integer scale of 1 (poor) to 5 (excellent) using the criteria in the rubric file named in the manifest. Write qualitative notes of at least 2-3 sentences per item. Check any flags that apply. You do not need to know the source of the content to score it — evaluate it on its scientific merits alone.

When finished, return the completed template to the study coordinator by email. Do not redistribute the blinded items or this template to others.
